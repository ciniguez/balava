JSTL: Instalación y Uso

Instalación

Para instalar JSTL lo puedes hacer de dos maneras

Opción manual

  • Descarga las librerías jstl-api.jar y jstl-impl.jar
    • O puedes descargar manualmente del repositorio de MAVEN
  • Copiar los Jar en la carpeta \WEB-INF\lib

Opción MAVEN

  • el fichero pom.xml, agrega una dependencia dentro del etiqueta <dependencies> (ver código ejemplo a continuación)
  • NOTA: También puedes visitar el repositorio MAVEN donde se indica el código de dependencia a incluir en el fichero pom.xml
<dependencies>
  	<dependency>
  		<groupId>javax.servlet</groupId>
  		<artifactId>jstl</artifactId>
  		<version>1.2</version>
  	</dependency>
  </dependencies>

Uso de JSP

  • Añadir
    • <%@ taglib prefix="c" uri="http://java.sun.com/jsp/jstl/core"%>
    • <%@ taglib prefix="fmt" uri="http://java.sun.com/jsp/jstl/fmt" %>
  • Se crea una variable: c:set (alcance: page, request, session, application)
    • <c:set var="p2" scope="request" value="${persona}" />
  • Expresiones (Expression Language o EL)
    • ${expresion}. Ejecuta la expresión
    • ${p1.id}. Invoca p1.getId()
    • ${p1.m()}. Invoca al método p1.m()
    • ${p1.id == false}. Realiza una comparación
  • Condicionales y bucles
    • Sentencia if: <c:if ...>
    • Sentencia else-if: <c:choose ...>
    • Sentencia for: <c:forEach ...>
