Configurando el Ambiente de Desarrollo

En el mercado existen popularmente dos opciones como IDE de desarrollo para Java Web, que se describen a continuación:
 
1.- NetBeans IDE
  • http://netbeans.org (7.3)
  • Versión *.exe, seguir los pasos de instalación
  • Incluye IDE, servidor JEE GlassFish y BD Apache Derby
2.- Eclipse IDE for Java EE Developers + Apache Tomcat + MySQL
  • Servidor Web: Apache Tomcat
    • http://tomcat.apache.org (8.0)
    • versión zip, sólo se descomprime.
    • Sólo esta certificado para servlets y JSP, el resto de JEE se deben instalar los Jars de la implementación manualmente.
  • Eclipse IDE for Java EE Developers
    • http://www.eclipse.org (ECLIPSE 2020-06)
    • Descomprimirlo y crear una carpeta para el “workspace”
El curso lo desarrollaremos con la segunda opción: Eclipse, Apache Tomcat y MySQL 

Actividad: Descargando e instalando lo necesario

  • Descargando e Instalando Apache Tomcat
    • Ingresa a la página http://tomcat.apache.org 
    • Descarga Apache Tomcat versión 8.0.x (versión zip)
    • Descomprime el zip descargado y ubícalo en un lugar del disco duro, por ejemplo C:/apacheTomcat (para windows) o /User/nombre_usuario/apacheTomcat (para MAC)
  • Descargando Eclipse
    • Ingresa a http://www.eclipse.org
    • Descarga la versión 2020-06
    • Descomprimir en un lugar del disco duro de preferencia
    • Ingresar al directorio descomprimido. Doble click en el lanzador de la aplicación
    • En la pantalla de selección y confirmación del Workspace, selecciona un directorio de tu disco duro que vayas a utilizar como Workspace.
